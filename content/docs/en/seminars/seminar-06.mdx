# Seminar 6 - Code Quality & Tests

## Lecture Topics

### Code Smells

#### Bloaters

- Classes or methods that are too large
- Hard to read and maintain

#### Long Method

- Method does too many things
- Split into smaller methods

#### Primitive Obsession

- Excessive use of primitive types
- Create custom classes

### Unit Tests

```java
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;

class PlayerTest {
    @Test
    void testAddScore() {
        Player player = new Player("Test");
        player.addScore(10);
        assertEquals(10, player.getScore());
    }

    @Test
    void testScoreCannotBeNegative() {
        Player player = new Player("Test");
        player.addScore(-5);
        assertEquals(0, player.getScore());
    }
}
```
