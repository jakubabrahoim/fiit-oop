# Seminar 2 - OOP Concepts

## Lecture Topics

### OOP Concepts

- **Class** - template/blueprint for objects
- **Instance** - concrete object created from a class
- **Package** - class organization
- **Method** - function belonging to a class

### 4 OOP Principles

1. **Encapsulation** - hiding implementation details
2. **Inheritance** - deriving classes
3. **Polymorphism** - multiple forms
4. **Abstraction** - simplifying complexity

### Access Modifiers

- `private` - accessible only within the class
- `public` - accessible from anywhere
- `protected` - accessible in package and subclasses
- (default) - accessible only within the package

```java
public class Animal {
    private String name;

    public Animal(String name) {
        this.name = name;
    }

    public String getName() {
        return name;
    }
}

public class Dog extends Animal {
    public Dog(String name) {
        super(name);
    }

    public void bark() {
        System.out.println(getName() + " says: Woof!");
    }
}
```
