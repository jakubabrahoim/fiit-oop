# Grading

## Points Overview

| Activity                | Points  | Notes                     |
| ----------------------- | ------- | ------------------------- |
| **Exam**                | 40      |                           |
| **Test**                | 10      | min. 4 points             |
| **Semester assignment** | 30      |                           |
| **Seminar activity**    | 10      |                           |
| **Small tests**         | 10      | (x/160)\*10               |
| **Total**               | **100** | min. 30 points for credit |

<Callout type='warn'>
    To earn credit, you need at least **30 points** and at least **4 points**
    from the test.
</Callout>

## Assignment Points Breakdown

| Part                               | Points | Requirements                                                                  |
| ---------------------------------- | ------ | ----------------------------------------------------------------------------- |
| **Specification and architecture** | 5      | Document: topic description, game type, simple UI sketch, list of Entities    |
| **Checkpoint**                     | 5      | Code + Presentation + Document: class diagram, OOP principles, 3-4 Unit tests |
| **Final submission**               | 20     | Code + Principles + Documentation + Tests + Presentation                      |
