# Course Information

## Assignment Topic

**Any computer game**

The assignment as a whole will be divided into 3 parts:

<Steps>
    <Step title='Specification'>
        Project intent, requirements, UML diagram, UI sketch through Shape-up
    </Step>
    <Step title='Checkpoint'>
        Implementation of basic functionality, OOP principles
    </Step>
    <Step title='Final submission'>
        Complete implementation - functional game, unit tests, JavaDoc, OOP
        principles, design patterns
    </Step>
</Steps>

## Requirements for Completion

To successfully obtain credit (be able to attent the final exam), it is necessary to:

- Earn at least **30 points** during the semester
- Earn at least **4 points** from the test
- Submit all parts of the assignment
