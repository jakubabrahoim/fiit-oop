# Informácie o predmete

## Téma zadania

**Ľubovoľná počítačová hra** 

Zadanie ako celok bude rozdelené na 3 časti: 

<Steps>
	<Step title="Špecifikácia">
		Zámer projektu, požiadavky, UML diagram, náčrt UI cez Shape-up
	</Step>
	<Step title="Checkpoint">
		Implementácia základnej funkcionality, OOP princípy
	</Step>
	<Step title="Finálne odovzdanie">
		Kompletná implementácia - funkčná hra, jednotkové test, JavaDoc, OOP princípy, návrhové vzory
	</Step>
</Steps>

## Podmienky absolvovania

Pre úspešné získanie zápočtu je potrebné:

- Získať minimálne **30 bodov** počas semestra
- Získať minimálne **4 body** zo semestrálneho testu
- Odovzdať všetky časti zadania
