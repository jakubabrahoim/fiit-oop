# Cvičenie 4 - Design Patterns

## Čo bolo na prednáške

### Návrhové vzory (Design Patterns)

#### Singleton

- Privátny konštruktor
- Jedna inštancia v celej aplikácii

```java
public class GameManager {
    private static GameManager instance;

    private GameManager() {}

    public static synchronized GameManager getInstance() {
        if (instance == null) {
            instance = new GameManager();
        }
        return instance;
    }
}
```

#### Observer

- Notifikácia o zmenách
- Publisher-Subscriber vzor

#### Strategy

- Výber algoritmu za behu
- Zapuzdrenie algoritmov

#### Decorator

- Rozšírenie funkcionality bez dedenia
- Wrapping objektov

### Kedy použiť `static` a `synchronized`

- `static` - zdieľané medzi všetkými inštanciami
- `synchronized` - thread-safe prístup

2. Aplikácia OOP princípov na hernú logiku
