# Cvičenie 5 - GUI & Plánovanie

## Čo bolo na prednáške

### Java Swing (základy)

- Grafické používateľské rozhranie
- Komponenty: JFrame, JPanel, JButton, JLabel
- Layout managers

```java
import javax.swing.*;
import java.awt.*;

public class GameWindow extends JFrame {
    public GameWindow() {
        setTitle("Moja Hra");
        setSize(800, 600);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        JPanel panel = new JPanel();
        panel.setLayout(new BorderLayout());

        JButton startButton = new JButton("Štart");
        panel.add(startButton, BorderLayout.SOUTH);

        add(panel);
        setVisible(true);
    }
}
```
