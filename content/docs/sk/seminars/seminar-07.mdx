### Switch vs. Polymorfizmus

**Zlý príklad:**

```java
switch (enemyType) {
    case "goblin": damage = 10; break;
    case "dragon": damage = 50; break;
    case "skeleton": damage = 15; break;
}
```

**Dobrý príklad:**

```java
interface Enemy {
    int getDamage();
}

class Goblin implements Enemy {
    public int getDamage() { return 10; }
}

class Dragon implements Enemy {
    public int getDamage() { return 50; }
}
```
