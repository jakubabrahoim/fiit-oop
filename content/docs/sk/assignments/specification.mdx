## Povolené frameworky

Pre vývoj hier môžete použiť tieto frameworky:

- [LibGDX](https://libgdx.com/) - Cross-platform game development framework
- [dyn4j](https://dyn4j.org/) - Physics engine for Java

## Úloha v rámci zadania

### 1. Zámer projektu

Opíšte:

- Aký typ hry budete robiť
- O čo v hre ide
- Kedy hráč vyhrá
- Herná mechanika
- Postavy
- Levely
- Objekty v hre

### 2. Funkcionálne požiadavky

- Spísať základné oblasti/celky, ktoré bude projekt mať
- K tomu napísať úlohy, ktoré treba vykonať
- Úlohy usporiadané podľa priority
- Môžete použiť Jiru alebo obdobný nástroj

<Callout type='info'>
    Nebudú to úplne funkcionálne požiadavky, skôr to bude backlog úloh, ktoré si
    máte spísať.
</Callout>

### 3. UML diagram tried

Vytvorte diagram najdôležitejších tried - tých, ktoré najviac vyjadrujú základnú myšlienku hry.

### 4. Nákres UI

Jednoduchý shape-up návrh podľa príkladu: [Shape Up - Chapter 2](https://basecamp.com/shapeup/1.1-chapter-02)

<Callout type='info'>
    Žiadne wireframes ani Figma nákresy. Má to byť jednoduchý náčrt.
</Callout>

## Dokumentácia musí obsahovať

1. ✅ Opis rámcovej témy
2. ✅ Akú hru budete robiť
3. ✅ Jednoduchý obrázok UI
4. ✅ Vymenovať Entity
